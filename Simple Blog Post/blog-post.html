CHESS BLOG

The game of chess is a very fun and exciting game. It needs a lot of practice to be able to master the game being that it is used for mental reasoning or development. It is a game in which only two persons can play at a time. Each player has sixteen(16) pieces each in which they use to defeat their opponent through checkmate.
